# Playbook de Onboarding Fiscal - Brasil (BR)

## Pre-requisitos
- CNPJ activo.
- Certificado Digital (Tipo A1).
- Inscripción Estadual / Municipal.
- Código de Servicio (CNAE).

## Pasos de configuración en Virteex
1. Navegar a **Admin > Configuración Fiscal**.
2. Seleccionar **País: BR**.
3. Cargar Certificado A1 (.pfx / .p12).
4. Configurar perfiles de impuestos (ICMS, IPI, PIS, COFINS).
5. Configurar serie y número de NF-e / NFS-e.

## Verificación de salida
- Transmisión a SEFAZ.
- Recibo de autorización.
- Impresión de DANFE.

## Resolución de problemas comunes
- Rejeição SEFAZ: Consultar código de error oficial en manual de SEFAZ.
- Certificado inválido: Verificar cadena de confianza de ICP-Brasil.
