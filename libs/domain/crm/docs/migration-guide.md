# CRM Domain Migration Guide

This guide describes the migration of the CRM domain to the v2.0 structure.

## Process
1.  **Documentation:** Added README, OWNERS, CHANGELOG and migration docs.
2.  **Domain Layer:** Reorganized entities and repository ports.
3.  **Application Layer:** Grouped use cases by Command/Query.
4.  **Contracts Layer:** Reorganized DTOs and Enums.
5.  **Infrastructure Layer:** Grouped persistence and integrations.
6.  **Presentation & UI:** Reorganized adapters and pages.
7.  **Enforcement:** Verified dependency boundaries.
