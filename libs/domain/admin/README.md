# Admin Domain

## Propósito
Dominio responsable de las operaciones administrativas globales de la plataforma, incluyendo la gestión de tenants, soporte técnico (incidencias) y configuración de la infraestructura.

## Perfil
- **Perfil:** `ui-domain`
- **Justificación:** El dominio de admin requiere una interfaz de usuario extensa para orquestar la configuración de otros dominios y supervisar el estado del sistema.

## Owners
- Equipo: Plataforma / SRE
